# 0001 - Document Architecture Decisions as ADRs

- Status: Accepted
- Date: 2026-04-22
- OverheadSeconds: 0

## Context

With frequent experimentation, decisions are easy to lose if they stay only in chat or commits.

## Decision

Store accepted and implemented architecture/technical decisions as ADRs in `docs/auto-doc/adr/`.
This decision is currently in effect in production/dev workflow.

## Consequences

- Better traceability and reviewability for active decisions.
- Small, ongoing documentation overhead.
