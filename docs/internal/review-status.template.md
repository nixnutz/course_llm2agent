# Local Review Status Template

Use this structure in your local (gitignored) `docs/internal/review-status.md`.

```md
## branch: <branch-name>
- last_reviewed_commit: <sha>
- last_reviewed_at: <iso-datetime>
- last_review_context: review-w-auto-doc
- last_action: reviewed | skipped_pending
- notes: <optional short note>
```
